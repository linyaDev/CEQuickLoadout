English
Tired of opening the CE loadout manager every time you want to add an item to a colonist's gear?

This mod adds a right-click menu when you select an item on the map. Quickly add items to colonist loadouts or create new loadouts — all without leaving the map view.

Features
- Select any item on the map, right-click to see loadout options
- "Add to colonist..." — submenu with all colonists, hover to see their current loadout
- "Create loadout" — creates a new loadout with the item and opens CE manager
- Works with Combat Extended Extended Loadout (PersonalLoadout support)

Requirements
- RimWorld 1.5
- Hardcore SK modpack
- Combat Extended
- Harmony

Русский
Надоело каждый раз открывать менеджер снаряжения CE чтобы добавить предмет колонисту?

Этот мод добавляет контекстное меню при правом клике на выделенный предмет на карте. Быстро добавляйте предметы в лоадауты колонистов или создавайте новые — не покидая карту.

Возможности
- Выделите предмет на карте, правый клик — опции лоадаутов
- «Добавить колонисту...» — подменю со списком колонистов, при наведении видно текущий лоадаут
- «Создать лоадаут» — создаёт новый лоадаут с предметом и открывает менеджер CE
- Совместимость с Combat Extended Extended Loadout (поддержка PersonalLoadout)

Требования
- RimWorld 1.5
- Модпак Hardcore SK
- Combat Extended
- Harmony
